# UTD - Projeto de Conversão de Bases com C
Universidade do Trabalho Digital - Projeto Conversor Bases com C

## Objetivo
Desenvolver um conversor de tipos de bases que suporte os tipos:
- Binário (2)
- Decimal (10)
- Hexadecimal (16)
- Octal (8)
